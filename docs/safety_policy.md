# 安全策略

## 允许的行为

- 读取离线日志、资产和任务数据。
- 生成飞行摘要、异常事件、故障假设和维护建议。
- 生成 Markdown 运维报告。
- 写入可审计记录。
- 提醒人工复核和批准要求。

## 禁止的行为

- 解锁或锁定真实飞行器。
- 启动电机。
- 执行起飞、降落、返航或航线飞行。
- 上传固件。
- 修改飞控关键参数。
- 向真实飞控发送命令。
- 绕过人工复核流程。

## Human-in-the-loop 要求

所有飞行关键、维护关键或参数相关建议都必须设置 `human_review_required=true`，并由具备资质的人员复核后才能在线下执行。

## 飞行关键动作限制

本项目不提供任何飞行关键动作接口。未来接入 MAVLink、PX4、ArduPilot 或 SITL 时，也必须先实现只读模式、仿真隔离和人工审批。
