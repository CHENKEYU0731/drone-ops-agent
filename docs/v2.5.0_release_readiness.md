## 安全边界

项目保持 offline-only、advisory-only 和 human-review-required。不连接真实无人机，不执行 MAVLink command、arm/disarm、takeoff、landing、RTL、mission execution、motor start、firmware upload 或 flight-controller parameter writing；不启动或连接 PX4、ArduPilot、Gazebo、SITL；不接入真实维修系统或 fleet platform。
