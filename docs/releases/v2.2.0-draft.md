# v2.2.0 - Evaluation and Case Study Baseline

Date: draft

Release status: draft

Release positioning: deterministic offline case-study evidence on top of the v2.1.0 demo and portfolio release.

## 主要变化

- 新增 `run-case-studies` CLI。
- 复用 14 个 offline/mock simulation matrix 场景和 diagnosis/report golden eval。
- 输出 `case_study_results.json`、中文 `case_study_report.md` 和 audit。
- 汇总预期状态准确率、证据覆盖率、误报数、漏检数和诊断/报告平均分。
- 使用固定时间戳、稳定排序和 canonical SHA-256 `result_digest` 保持确定性。
- 将案例研究纳入一键演示成果包。
- README 增加 3 分钟快速体验、架构流程和真实运行截图。
- 演示 Dashboard artifact refs 改为可移植相对路径。
- 项目版本更新为 `2.2.0`。

## 验证命令

```bash
pytest
python scripts/generate_demo_outputs.py --out demo_outputs
python -m apps.cli.main run-case-studies --simulation-matrix data/sample_simulation/scenario_matrix.json --eval-case data/sample_evals/diagnosis_report_eval_case.json --out <tmp-case-study-dir>
python -m compileall -q apps packages scripts
```

## 安全边界

本版本保持 offline-only、advisory-only 和 human-review-required。不调用外部模型，不连接真实无人机、飞控、仿真器、维修系统或 fleet platform，不执行 MAVLink command，不自动派单，不上传固件，也不写入飞控参数。
