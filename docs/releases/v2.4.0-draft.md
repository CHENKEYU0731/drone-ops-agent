# v2.4.0 - Reproducible Distribution

Date: draft

Release status: draft

## 主要变化

- 新增稳定 JSON 环境诊断，不记录本地路径或凭据。
- 固定核心、开发、可选 parser 和发布工具的直接依赖版本。
- 新增 Windows 临时虚拟环境发布验收脚本。
- 验收覆盖全量测试、demo、wheel/sdist 构建、wheel 安装和 CLI smoke test。
- 新增固定文件顺序、时间戳、权限和内容 manifest 的确定性 ZIP。
- 发布包同时提供内部 `SHA256SUMS` 和外部 ZIP SHA-256。
- 项目版本更新为 `2.4.0`。

## 安全边界

依赖安装属于显式发布操作；项目运行时仍为 offline-only、advisory-only 和 human-review-required。不连接真实无人机、飞控、仿真器、维修系统或 fleet platform，不执行 MAVLink command，不自动派单，不上传固件或写入飞控参数。
