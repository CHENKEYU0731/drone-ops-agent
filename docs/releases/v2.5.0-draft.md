# v2.5.0 - Portfolio Finalization

Date: draft

Release status: draft

## 主要变化

- 新增中文最终项目总览和英文项目摘要。
- 新增覆盖输入、输出、证据和边界的能力矩阵。
- 新增 3–5 分钟演示脚本和中英文项目说明。
- 新增 sanitized 最终作品展示包生成脚本。
- 展示包包含现有 demo 成果、说明文档、截图、每文件 SHA-256 manifest、ZIP 和外部校验文件。
- 展示包拒绝外部 `.ulg`/`.bin`、本地仓库路径和未受管理目录覆盖。
- PDF 导出启用 invariant 模式，提高同输入 PDF 的可重复性。
- 明确 golden/mock 指标和上游 fixture 兼容性都不代表真实世界准确率。
- 项目版本更新为 `2.5.0`。

## 安全边界

最终作品展示仍为 offline-only、advisory-only 和 human-review-required。不连接真实无人机、飞控、仿真器、维修系统或 fleet platform，不执行 MAVLink command，不自动派单，不上传固件或写入飞控参数。
