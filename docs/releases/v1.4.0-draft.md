# v1.4.0 - Diagnosis and Report Evaluation

Date: draft

Release status: draft

Release positioning: local diagnosis/report evaluation quality gate on top of the v1.3.0 rule pack and skill versioning release.

## 范围

v1.4.0 新增诊断与报告评估 baseline，用于把诊断假设、维修建议、证据引用、报告章节和安全边界声明纳入本地 golden eval 检查。

该版本仍然是 offline-only、advisory-only 的质量门禁版本，不是飞控集成版本，不是外部模型评测平台，也不是维修系统集成版本。

## Eval Contract Baseline

新增能力：

- 新增 `EvalStatus`。
- 新增 `EvalMetric`。

- 新增 `EvalCase`。
- 新增 `EvalResult`。

## 安全边界

v1.4.0 不包含：

- 真实无人机连接
- MAVLink command execution
- arm/disarm
- takeoff / landing / RTL
- mission execution
- motor start
- firmware upload
- flight-controller parameter writing
- PX4、ArduPilot、Gazebo、SITL 或外部仿真器启动或连接
- 外部模型调用
- 真实 fleet platform API 调用
- 真实 CMMS、Jira、飞书、企业微信或其他维修系统 API 调用
- 自动派单
- 自动执行维护动作
- 真实、敏感或未批准的二进制飞行日志

所有 eval 结论都默认 `human_review_required=true`，只用于本地质量门禁和人工复核材料。
