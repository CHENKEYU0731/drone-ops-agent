# v0.8.0 - Report Integration Plus

Date: Draft

Release status: draft

Release positioning: offline report integration improvements on top of the
v0.7.0 quality gates.

## 范围

v0.8.0 计划把日志解析、异常、诊断、维护建议、离线仿真验证和审计信息更完整地纳入单次运维复盘报告。

本版本仍然是 offline-only、mock-import-only、advisory-only 的报告增强版本，不是飞控集成版本。

## PR 1: Report Simulation Validation Section Baseline

第一个 v0.8.0 PR 让 `generate-report` 可以选择性纳入离线仿真验证结果。

新增能力：

- `generate-report --simulation <simulation_run.json>`
- Markdown 运维报告新增 `## 7.5 仿真验证` 章节。
- 报告展示 scenario ID、仿真状态、人工复核要求、规则命中详情和 evidence refs。
- 证据引用附录会纳入 `simulation_run.json` 的仿真证据。
- `ops-report-generation` audit input refs 会记录传入的 `simulation_run.json`。

兼容性：

- 未提供 `--simulation` 时，现有报告生成行为保持不变。
- 新章节插入在安全说明之前，不改变现有 `## 10. 审计记录` 和 `## 11. 证据引用附录` 标记。

## PR 2: Report Audit Metadata and Review Checklist Baseline

第二个 v0.8.0 PR 让 Markdown 运维报告更适合作为人工复盘材料。

新增能力：

- 报告新增 `## 7.6 审计摘要`。
- 报告新增 `## 7.7 日志解析元数据`。
- 报告新增 `## 7.8 人工复核清单`。
- `generate-report` 会读取同一输出目录下已有的 `audit/*.json`，并按稳定顺序纳入报告。
- 日志解析元数据展示 requested format、actual format、parser name、parser version、warnings 和 parser metadata。
- 人工复核清单覆盖异常、故障假设、维护建议和可选离线仿真结论。

兼容性：

- 不新增 CLI 参数。
- 不改变现有结构化 JSON 输出。
- 不改变 `validate-report` 需要识别的 `## 10. 审计记录` 和 `## 11. 证据引用附录` 标记。

## PR 3: Report PDF Section Coverage Baseline

第三个 v0.8.0 PR 确认 `generate-report --pdf` 会同步包含 v0.8.0 新增报告章节。

覆盖内容：

- `## 7.5 仿真验证`
- `## 7.6 审计摘要`
- `## 7.7 日志解析元数据`
- `## 7.8 人工复核清单`
- 仿真规则 evidence refs，例如 `SIM_RESULT_COMPLETED`
- parser metadata，例如 `csv-json-flight-log@1.1.0`

本 PR 不重写 PDF exporter，不引入高级排版，只补充 PDF 文本提取验证，确保新增 Markdown 章节不会在 PDF 输出路径中回归。

## 路线图整理

本 PR 同时把根目录中的中文规划文本整理为 Markdown 文档：

- `docs/roadmap.md`
- `docs/planning/v0.8-to-v2.0-roadmap.md`

如继续扩展 v0.8.0 后续补丁，建议仍保持小范围、可验证，并避免进入 v0.9.0 工单草稿或 v1.x 平台化能力。

## 验证计划

本 PR 至少需要通过：

- `pytest`
- focused report / CLI / validation tests
- `run-mvp` 生成临时报告目录
- `validate-simulation` 生成离线/mock `simulation_run.json`
- `generate-report --simulation <simulation_run.json>` 生成包含仿真验证章节的 Markdown 报告

## 安全边界

v0.8.0 继续保持离线建议系统定位。

本版本不包含：

- 真实无人机连接
- MAVLink command execution
- arm/disarm
- takeoff / landing / RTL
- mission execution
- motor start
- firmware upload
- flight-controller parameter writing
- 真实 PX4、ArduPilot、Gazebo、SITL 或外部仿真器启动或连接

所有报告、仿真验证和维护建议仍必须由人工复核，不代表真实飞行授权。
