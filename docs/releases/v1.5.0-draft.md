# v1.5.0 - Platform Readiness

Date: draft

Release status: draft

Release positioning: local platform-readiness contracts and governance baseline on top of the v1.4.0 diagnosis/report evaluation release.

## 范围

v1.5.0 为后续 v2.0.0 平台化准备本地工程边界：workspace/project、report bundle manifest、reviewer / approval model、offline adapter contract、数据保留与脱敏治理清单。

该版本仍然是 offline-only、advisory-only 的本地质量门禁版本，不是外部平台集成版本，不是真实维修系统集成版本，也不是飞控集成版本。

## Platform Contract Baseline

新增能力：

- 新增 `WorkspaceProject`。

- 新增 `ReviewerApproval`。

## 安全边界

v1.5.0 不包含：

- 真实无人机连接
- MAVLink command execution
- arm/disarm
- takeoff / landing / RTL
- mission execution
- motor start
- firmware upload
- flight-controller parameter writing
- PX4、ArduPilot、Gazebo、SITL 或外部仿真器启动或连接
- 真实 fleet platform API 调用
- 真实 CMMS、Jira、飞书、企业微信或其他维修系统 API 调用
- 自动派单
- 自动执行维护动作
- 真实、敏感或未批准的二进制飞行日志

所有平台准备输出都默认 `human_review_required=true`，只用于本地质量门禁和人工复核材料。
