# v1.3.0 - Rule Pack and Skill Versioning

Date: draft

Release status: draft

Release positioning: offline rule governance and skill version metadata baseline on top of the v1.2.0 local dashboard release.

## 范围

v1.3.0 引入规则包、规则版本和 skill registry 的本地治理能力。它仍然只是 offline-only、
advisory-only 的审计与验证增强，不是飞控执行系统，也不是外部平台集成。

## Rule Pack Contract Baseline

新增能力：

- 新增 `RulePackScope`。
- 新增 `RulePackRule`。
- 新增 `RulePack`。
- 新增 deterministic sample rule pack：`data/sample_rule_packs/offline_default_rules.json`。
- 新增 `docs/rule_pack_versioning.md`。

## 后续 v1.3.0 小阶段

1. Rule pack validation CLI。
2. Skill version registry and v1.3.0 readiness。

## 安全边界

v1.3.0 不包含：

- 真实无人机连接
- 实时遥测连接
- MAVLink command execution
- 真实 PX4、ArduPilot、Gazebo、SITL 或外部仿真器启动或连接
- 真实 fleet platform API 调用
- 真实 CMMS、Jira、飞书或企业微信 API 调用
- 自动派单
- 自动执行维护动作
