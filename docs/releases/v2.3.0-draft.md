# v2.3.0 - Open-Source Upstream Log Compatibility

Date: draft

Release status: draft

## 主要变化

- 新增固定 commit、许可证、大小和 SHA-256 的公开 ULog 注册表。
- 外部获取与项目运行时严格分离，项目不内置网络客户端，缓存目录不提交。
- 新增注册表验证和离线上游日志案例研究 CLI。
- PX4 parser 支持上游 `output[0..3]`、PWM、barometer temperature、multi-id topic 和丢包率字段。
- 明确记录启动相对时间、actuator 映射和链路字段限制。
- 当前 3 个 BSD-3-Clause 上游 fixture 均解析成功。
- 明确 `real_world_flight_verified=false`，不将兼容性结果包装成真实准确率。
- 项目版本更新为 `2.3.0`。

## 安全边界

外部获取必须由用户使用浏览器或系统工具显式执行。项目运行时代码不包含网络客户端；分析保持 offline-only、advisory-only 和 human-review-required。不连接真实无人机、飞控、仿真器、维修系统或 fleet platform，不执行 MAVLink command，不自动派单，不上传固件或写入飞控参数。
