# v2.1.0 - Demo and Portfolio Readiness

Date: draft

Release status: draft

Release positioning: repeatable offline demo and portfolio package on top of the v2.0.0 operations platform baseline.

## 范围

v2.1.0 提供一条可重复运行的本地演示链路，用仓库内 sample / mock / sanitized 数据生成报告、PDF、证据索引、仿真验证、工单草稿、机队健康、Dashboard 数据包和平台验证结果，便于项目理解、导师检查和作品展示。

## 主要变化

- 新增 `scripts/generate_demo_outputs.py`，生成受管理的本地成果包。
- 新增中文演示说明 `docs/demo_guide.md`。
- 使用 `.drone-ops-demo-output` 标记受管理输出，拒绝危险路径和未受管理的非空目录。
- 强化 platform index 与 operations platform baseline 的模块和产物引用检查。
- `REVIEW_REQUIRED` 平台验证结果使用非零 CLI 退出码并写入对应 audit 状态。
- preflight `GO` 和所有关键演示结论继续保持 `human_review_required=true`。
- 项目包版本更新为 `2.1.0`。

## 验证命令

```bash
pytest
python scripts/generate_demo_outputs.py --out demo_outputs
python -m compileall -q apps packages scripts
```

详细检查项见 `docs/v2.1.0_release_readiness.md`。

## 安全边界

本版本保持 offline-only、advisory-only 和 human-review-required。不连接真实无人机，不执行 MAVLink command，不启动或连接 PX4、ArduPilot、Gazebo、SITL，不接入真实维修系统或 fleet platform，不自动派单，不上传固件或写入飞控参数。
