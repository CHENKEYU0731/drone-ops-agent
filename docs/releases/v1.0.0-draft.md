# v1.0.0 - Stable Offline Ops Agent

Date: draft

Release status: draft

Release positioning: stable offline advisory baseline covering the accumulated
capabilities from `v0.1.0` through `v0.9.0`.

## 范围

v1.0.0 不是新增大功能版本，而是把此前从 `v0.1.0` 到 `v0.9.0` 形成的离线无人机运维
Agent 能力收成第一个可靠主版本。

本版本聚焦：

- CLI contract freeze
- schema contract freeze
- audit contract freeze
- release readiness checklist
- safety boundary regression
- 文档状态同步

## Stable Contract Inventory

新增文档：

- `docs/v1.0.0_stable_contract.md`

该文档冻结 v1.0 稳定契约，包括：

- 稳定 CLI 命令清单
- 稳定 JSON 输出清单
- 稳定报告输出清单
- 稳定 schema surface
- 稳定 audit contract
- offline-only / advisory-only 安全边界
- v1.0 之后仍属于 experimental / future extension 的能力

## Safety Regression Gate

新增文档和测试：

- `docs/v1.0.0_safety_regression_gate.md`
- `tests/unit/test_v1_safety_regression_gate.py`

该门禁用于确认 v1.0.0 收口阶段没有引入真实控制路径或真实外部系统调用。它聚焦
`apps/` 和 `packages/` 下的 runtime code，检查没有 MAVLink command execution、
flight-controller parameter writing、真实仿真器启动、真实网络 client 或真实维修系统集成。

文档和测试中可以继续出现安全边界相关术语，用于说明禁止项和验证策略；runtime code
不得新增真实控制、真实仿真器或真实工单系统调用能力。

## 安全边界

v1.0.0 继续保持离线建议系统定位。

本版本不包含：

- 真实无人机连接
- MAVLink command execution
- arm/disarm
- takeoff / landing / RTL
- mission execution
- motor start
- firmware upload
- flight-controller parameter writing
- 真实 PX4、ArduPilot、Gazebo、SITL 或外部仿真器启动或连接
- 真实 CMMS、Jira、飞书或企业微信 API 调用
- 自动派单
- 自动执行维护动作
- 真实、敏感或未经确认允许的二进制飞行日志
